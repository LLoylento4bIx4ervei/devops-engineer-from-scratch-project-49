**Brain Games** — набор из пяти консольных игр для тренировки ума. Каждая игра задаёт три вопроса, и для победы нужно ответить правильно на все три.

### Игры

| Команда | Описание |
|---------|----------|
| `brain-even` | Показано число, нужно ответить `yes`, если чётное, иначе `no`. |
| `brain-calc` | Показано арифметическое выражение, нужно вычислить результат. |
| `brain-gcd` | Показаны два числа, нужно ввести их наибольший общий делитель. |
| `brain-progression` | Показана арифметическая прогрессия с пропущенным числом, нужно его найти. |
| `brain-prime` | Показано число, нужно ответить `yes`, если оно простое, иначе `no`. |

## Установка и запуск

```bash
git clone https://github.com/LLoylento4bIx4ervei/devops-engineer-from-scratch-project-49.git  
cd devops-engineer-from-scratch-project-49  
uv build  
uv tool install dist/*.whl  

После установки любая игра запускается командой:  

brain-even  
brain-calc  
brain-gcd  
brain-progression  
brain-prime
